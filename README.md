## Quick Start

From the repository root:

1. Initialize submodules:
	- `git submodule sync`
	- `git submodule update --init --recursive`
2. Enter a project folder under `week1/`
3. Install dependencies and run tests (example):
	- `npm install`
	- `anchor test`
